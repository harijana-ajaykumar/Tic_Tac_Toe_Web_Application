# Tic Tac Toe Web Application

## Overview
The Tic Tac Toe Web Application is a fun and interactive browser-based game developed using HTML, CSS, and JavaScript. This project recreates the classic Tic Tac Toe game where two players take turns marking X and O on a 3×3 grid. The application automatically checks for winning combinations, detects draws, and allows players to restart the game at any time.

## Features
- Interactive 3×3 game board
- Two-player gameplay
- Automatic winner detection
- Draw detection
- Restart game functionality
- Responsive and user-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Use
1. Open the application in a web browser.
2. Player X starts the game.
3. Players take turns selecting empty cells.
4. The game automatically announces the winner or a draw.
5. Click the restart button to play again.
