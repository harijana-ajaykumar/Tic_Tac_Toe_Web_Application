# Tic Tac Toe Web Application

## Description
This is a simple Tic Tac Toe web application developed using HTML, CSS, and JavaScript. The game allows two players to play alternately on a 3x3 grid. The application automatically detects the winner and provides an option to restart the game.

## Features
- Interactive 3x3 game board
- Two-player gameplay (X and O)
- Winner detection
- Draw detection
- Restart game option
- Responsive and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.
